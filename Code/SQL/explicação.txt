# Explicação dos Dados Inseridos no Banco TGControl (`inserts.sql`)

Este documento descreve a estrutura e os cenários representados pelos dados de exemplo inseridos no banco de dados `TGControl`. Todos os usuários criados possuem a senha `Troca123`.

## 1. Estrutura Geral

Foram criados:
* **5 Professores:**
    * 3 Orientadores (`orientador.silva`, `orientador.costa`, `orientador.lima`).
    * 2 Coordenadores de TG (`coord.alves`, `coord.borges`).
* **3 Turmas (`class`):**
    * `Banco de Dados` (BD), início 2025/1.
    * `Analise e Desenvolvimento de Sistemas` (ADS), início 2025/1.
    * `Educacao Ambiental` (EA), início 2024/2.
* **30 Alunos:**
    * Os 7 membros da equipe original + 23 alunos genéricos (`alunoXX.gen`).
    * Distribuídos entre as 3 turmas.
    * Com orientadores variados (alguns `NULL`).
    * Em diferentes estágios do TG (`estagio_tg_atual` 1 = TG1/5º Sem., 2 = TG2/6º Sem.).
* **Tarefas (`task`):** 180 tarefas (6 para cada um dos 30 alunos). O template padrão é:
    1.  `Apresentação Pessoal e Acadêmica`
    2.  `Relatório PIM II`
    3.  `Relatório PIM III`
    4.  `Relatório PIM IV`
    5.  `Relatório PIM V`
    6.  `Relatório PIM VI`
* **Submissões (`task_submission`) e Revisões (`task_review`):** Exemplos para alguns alunos/tarefas, mostrando o fluxo de envio, pedido de revisão e aprovação.
* **Notificações (`notification`):** Exemplos para alguns eventos de submissão/revisão, algumas marcadas como lidas (`is_read=1`) e outras não (`is_read=0`).
* **Coordenação (`tg_coordenacao_turma`):** Atribuição de responsabilidade de etapa para os Coordenadores TG em cada turma.
* **Defesa (`defesa_tg`):** A tabela está vazia por padrão, a ser preenchida quando um professor agendar a defesa de um aluno concluído.

## 2. Perfis Principais e Cenários

### Professores

* **Orientadores (Silva, Costa, Lima)**: Professores padrão que orientam alunos (`advisor_email` na tabela `student`). Podem revisar e aprovar/solicitar revisão das submissões (`task_review`).
* **Coordenador Alves (`coord.alves@...`)**: Professor de TG (`is_coordinator=1`). Responsável pela **Etapa 1** (Tarefas 1-4) das turmas de BD 2025/1 e ADS 2025/1.
* **Coordenador Borges (`coord.borges@...`)**: Professor de TG (`is_coordinator=1`). Responsável pela **Etapa 2** (Tarefas 5-6) das turmas de BD 2025/1 e ADS 2025/1. Também é responsável por **ambas as etapas (1 e 2)** da turma de EA 2024/2.
